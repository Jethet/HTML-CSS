Using the different tags shows the differences in size of headings


Available for everyone, funded by readers (this is a blockquote)

To show the various options, here we write in strong and after that we use dfn as another tag, which marks the introduction of a new word. Then there is this b tag that adds emphasis. The same can be done using the elements that are referred to as em and i tags, or the mark tag. Although highlighting can be achieved with CSS, using the mark tag gives an indication to screen readers and web crawlers that this is important. One last element is the q element, which automatically inserts quotation marks around the text inside of the tags.

All these tags are part of semantic HTML.
